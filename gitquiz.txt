1. Git clone.
2. Change the directory from home to your repository folder.                	<cd justin-1948-cmis>
3. Create a new text file called file.txt in your repository folder.         	<gedit file.txt>
4. Save the file and close gedit.
5. Git add your work back to github.com. <git add file.txt>
6. Git commit your work back to github.com. <git commit -m "Commit Comment">
7. Git push your work back to github.com. <git push>
8. Enter your username and password for git push.
9. Check if the file is in your respository in github.com.
10. Close terminal and shut down your computer.
